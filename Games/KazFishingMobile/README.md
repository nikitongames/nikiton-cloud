# KazFishing Mobile
Мобильная версия симулятора рыбалки от NikitoN Games.  
Проект построен на NikitoN Core Framework и использует общие модули ядра.

## Подключённые модули:
- Configs  
- Engine  
- Monetization  
- Policies  
- Templates  

## Тип проекта:
Fishing Simulation  
С поддержкой системы погоды и динамических локаций.
