Nikiton Core Framework 2.0
--------------------------
Описание:
Универсальное ядро для всех проектов NikitoN Games.
Обеспечивает:
 - Обновления через Nikiton Cloud
 - Проверку устройства и соглашений
 - Экран загрузки с логотипом
 - Поддержку валют, магазина, событий, туров
 - Модули состязаний, турниров и индикаторов

Версия: 2.0.0
Автор: NikitoN Games
Дата: 2025
